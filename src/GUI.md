# 图形化界面

- **Tips-86:** 进入和退出图形化调试界面

**示例代码：**

```c
#include <stdio.h>

void fun1(void)
{
        int i = 0;

        i++;
        i = i * 2;
        printf("%d\n", i);
}

void fun2(void)
{
        int j = 0;

        fun1();
        j++;
        j = j * 2;
        printf("%d\n", j);
}

int main(void)
{
        fun2();
        return 0;
}
```

- **Tips-89:** 调整窗口大小

**示例代码：**

```c
#include <stdio.h>

void fun1(void)
{
        int i = 0;

        i++;
        i = i * 2;
        printf("%d\n", i);
}

void fun2(void)
{
        int j = 0;

        fun1();
        j++;
        j = j * 2;
        printf("%d\n", j);
}

int main(void)
{
        fun2();
        return 0;
}
```

   │20      }                                                                                      │

   │23      {                                                                                      │
   │24              fun2();                                                                        │

   │                                                                                               │
